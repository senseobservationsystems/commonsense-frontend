Welcome to your data!

Here you can find and compare all data you have collected in CommonSense.

Adding data

If you don't have any data sources connected yet the easiest way to get started is by installing one of the phone apps. Use the links below or search for "Sense Platform".

The sidebar

← This sidebar shows all data streams which are available to you. The sidebar has 4 sections:

  • Sensor library: This lists all available sensors, physical as well as virtual.
  • Manage group memberships: Join a group to share data or to make use of data shared by others
  • Manage states: This advanced feature lets you add state sensors.
  • Manage environments: This sections allows you to indicate that one or more sensors belong to a certain location.

This panel

As soon as you have any sensors in your library, you can drag them to this panel to visualize them.

Comparing sensors

You can even select multiple sensors, by holding [cmd] or [ctrl]. Drag a selection to this panel and you'll be able to compare them!
